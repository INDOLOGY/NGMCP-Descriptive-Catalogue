Manuscript Features

This MS consists of Dharmadāsa’s commentary on Candragomin’s sūtras 2.2.1 up to 2.2.87, some words of the section on 2.2.1 being missing. Thus, fols. 70-83 are extant, the copyist having skipped number “82”, counting “81”, “83”.

Exp. 2 shows a fol. possibly used as a cover leaf, which originally might have been part of the MS, but the script seems to be blurred because of some problem in the process of micro-filming.
