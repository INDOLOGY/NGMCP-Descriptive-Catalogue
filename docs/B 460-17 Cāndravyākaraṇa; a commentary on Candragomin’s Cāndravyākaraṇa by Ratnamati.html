Manuscript Features

  • The available folios are 2–19 and 27–34.
  • The text ends with the 3rd pāda of the 5th adhyāya.
  • The colophon names not only the scribe, Yajñānanda Vajrācārya, but also the corrector, Rāmacandra Śarmā.

Beginning
