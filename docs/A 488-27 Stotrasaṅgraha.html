Manuscript Features

The MS contains the following texts: 1. Dakṣiṇakālikākavaca 2. Viṣṇoḥ Prāṇaprasamanastotra 3. Chinnamastākavaca 4. Viṣṇupañjarakavaca 5. Karpūrastotra 6. Bhagavadgītāyāḥ Daśamodhyāyaḥ 7. Sūryanamaskāraphalastuti 8. Kārttavīryārjunakavaca 9. Viṣṇor nāmāṣṭakastotra 10. Nirañjanātmakagoraṣa-īśvarasya sahasranāmaṣoḍaśapaṭala
