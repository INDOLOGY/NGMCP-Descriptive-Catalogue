Manuscript Features

This MTM contains the following texts:

1. Sāṅkhyatattvakaumudī (fol. 1–48, exps. 3t–47t)

2. Sāṅkhyakārikā (fols. 2–5, exps. 47b–51t)

3. Unknown portion of the Mahābhārata (fol. 42, exps. 51b-52)

Fols. 5, 6 and 27 of the first text are missing.
