Manuscript Features

MS contains the Scattered folios 113r–115v of the Pratiṣṭhāvidhi, which contains the

chapters of kūpataḍāgapuṣkariṇipratiṣṭhā and prāsādapratiṣṭḥāvidhiḥ

MS is filmed in reverse order.
