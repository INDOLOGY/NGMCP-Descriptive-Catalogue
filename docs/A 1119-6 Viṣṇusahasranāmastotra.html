Manuscript Features

Fols. 63–65 are missing.

The foliation done in the upper left-hand margin of the verso is added later. It starts from one and runs so forth.

The foliation of the middle right-hand margin of the verso is actual foliation done by scribe. It starts from fifty six and runs so forth. So that we can guess that there were other texts before this.

Excerpts

Beginning

❖ oṁ namo bhagavate vāsudevāya⟨ḥ⟩ ||    ||

viṣṇusahasranāma⟨ṃ⟩ ||

śuklāṃbaradharaṃ viṣṇuṃ śaśivarṇṇaṃ caturbhujaṃ |

prasannavadanaṃ dhyāye,t sarvvavighnopaśāntaye ||

brahmalokād iha prāptaṃ, nāradaṃ bhagavatpriyaṃ |

dṛṣṭvā natvā sabhāmadhye, prapa⟨kṣu⟩[cchu]r ma(!)nayo⟨r⟩ ⟨m⟩mudā (fol. 56r1–2; (1v1–2, this foliation is added later))

End

śrīīśvara uvāca |

rāma rāmeti rāmeti, rame rāme manorame |

sahasranāmabhis tulyaṃ, rāma nāma varānane || (fol. 73v5–6; (fol. 18v5–6, this foliation is added later))

Colophon

iti śrīpadmapurāṇe umāmaheśvarasaṃvāde viṣṇo[ḥ] sahasranāmastotraṃ samāptaṃ (fol. 73v6; (18v6, this foliation is added later))
