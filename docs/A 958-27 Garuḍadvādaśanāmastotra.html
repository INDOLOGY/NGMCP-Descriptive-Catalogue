Manuscript Features

A picture of śrīyantra on fol. 1v, exp. 4.

The missing folios are 62–63.

The preliminary database doesn’t mark it as a MTM but in the manuscript, more MTM texts are mixed with single texts.

At the end of the manuscript, there is an unclear text.
