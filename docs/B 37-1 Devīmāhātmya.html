Manuscript Features

Fol. 1v is awfully damaged loosing very large part of the text.

There is some text written on the front cover-leaf. Since it is badly faded out, the following is what we are able to read:

vāgīśvarāya namaḥ

ca[ṇḍī]pāṭha


On fol. 31v, above and below the line of the colophon, there is some text written in a different hand, beginning with '❖ śrīvandanā
