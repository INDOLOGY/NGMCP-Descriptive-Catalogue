Manuscript Features

On exp. 2, a few lines about the Dharmaśāstra are available.

MS contains the chapter Kukarmajavipākaśuddhiprāyaścittavidhi assigned to the Saddharmatripiṭaka.

lokasya duḥkhārṇavaśoṣaṇan tam |

na praṇaśyanti karmāṇi kalpakoṭiśatair api |

sāmagrīprāpya kālañ ca phalanti khalu dehinām || (fol. 1v1–4)
