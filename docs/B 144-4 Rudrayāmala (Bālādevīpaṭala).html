Manuscript Features

The MS is in modern book form. It contains the following sub-chapters of Bālādevīpaṭala: Bālātripurasundarīnityārcana (fols. 1–11), Bālātripurasundarī¬sahasranāmastotra (fols. 1–10), Bālātripurasundarīkavaca (fols. 1–3) and Bālātripurasundarīstotra (fols. 1–2).

The foliation is irregular. Sometimes it appears on both the left- and right-hand pages. It restarts from each new sub-chapter, and in cases where the new sub-chapter begins on the same page where the preceding sub-chapter has ended, it will have two different page numbers.

On exp. 2 is written: dhūpamudrā tarjanyaṃguṣṭhayogena dīpa madhyamāṃguṣṭhayogena naivedya aṅguṣṭhānāmikāyogena …

There are two exposures of fols. 4–5 (last foliation).
