Manuscript Features

This manuscript does not contain any text. It has not been mentioned in the PTL as well. It bears only two cover-leaves. In the left-hand part of exp. 2 is written in a different hand: Śītalāko 1 patra. By this we can say that it might be a fragmentary folio of certain Śītalāstotra. The exp. 2 contains four lines where as it is completely illegible and does not look like a beginning of the text or part of the text. It does not seem that it is an important fragmentary folio of some certain text. So, we are mentioning this much information here to remove the confusion for the scholars, who are interested to see the Reel No. B 37/61.
