Manuscript Features

There is an extra folio to this MS (exp. 2–3) serving as a cover leaf. The hand is similar to the scribe of the text proper, the language is equally Newari. Another folio has been attached to the end of this MS (exp. 26–27).

The writing on fols. 5v–6r has been partly rubbed off.
