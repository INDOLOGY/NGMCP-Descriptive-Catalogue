Excerpts

Beginning of the root text

ayaṃ pārāśarī horām anusmṛtya yathāvidhi
budhair mānādibhis sarvai(2)r yathā sā[[mā]]nyaśāstrataḥ
etac chāstrānusāreṇa saṃjñāṃ vrūmo viśeṣataḥ 3 (fol. 2r1–2)

Beginning of the commentary

-tahoṃ jyotiṣin ke harṣake vāste
(2) ora nakṣatradaśā prakāra karī(3)ke phalakahaneko prakāra kahatu ho ora bhāvako ādi leke jo sava vicāraheṃ so sāmānyaśāstra(4)seṃ bṛhajjātakaseṃ jānanā ava isa śāstrake anusāraseṃ saṃjñāviśeṣaka karikaiṃ kahatu ho 3 (fol. 2r1–4)

End of the root text

parasparadaśāyāṃ tu bhuktau sūrya (9) ya bhārgavau (!) ||
vyatya yena viśeṣeṇa pradiśetāṃ śubhāśubhaṃ || 39 || (fol. 12v8–9)

End of the commentary

śukra śanaiścara je dono graha hai so paraspara da(10)śāke viṣe apanī apanī antardaśā meṃ vatyaya (!) kari ke śubhāśubhaphalakoṃ dete hai ye śukra jo hai so śanaiśca(11)rakī daśāmeṃ java śukra kī antardaśā āve usa antardaśā meṃ śaniścarake śubhāśubhaphalako śu (fol. 13v9–11)
