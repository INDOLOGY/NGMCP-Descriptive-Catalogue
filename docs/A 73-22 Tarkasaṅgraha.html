Manuscript Features

This MS contains both the Tarkasaṃgraha and Annambhaṭṭa’s autocommentary on it, namely the Tarkasaṃgrahadīpikā. The commentary is written above and below the basic text on each folio. The MS is dated śaṃvat 1864 āṣāḍhamāseśuklapakṣe saptmyāṃ ravivāsare.
