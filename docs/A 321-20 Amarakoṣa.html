Manuscript Features

adhigaganam anekā (!) tārakā rājyam āpuḥ

pratigṛham api dīpāḥ prāpnuvanti prasutvaṃ (!) ||

diśi diśi vikasantaḥ sa(2)nti khadyotapotā[[ḥ]]

savitari paribhūte kin na (!) lokair aloki || 1 || etc.

There are many footnotes in the margins.

There are three kāṇḍas of this text, but the manuscript contains only first and second kāṇḍas.

Foliation starts newly in each chapter.

samāhṛtyānyataṃtrāṇi (2) saṃkṣiptaiḥ pratisaṃskṛtaiḥ ||

prāyaśo rūpabhedena sāhacaryyāc ca kutracit ||

strīpuṃnapuṃ(3)sakaṃ jñeyaṃ tadviśeṣavidheḥ kvacit ||
