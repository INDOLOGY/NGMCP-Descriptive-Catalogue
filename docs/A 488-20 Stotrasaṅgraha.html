Manuscript Features

This MS contains the following texts: 1. Pūjāvidhi 2. Turaṇīstava 3. Pāṇḍavagītāstotra 4. Viṣṇubhujaṅgaprayāta written by Śaṅkarācārya 5. Rāmacandranāmāṣṭottaraśata

The first text Pūjāvidhi is in exp. 3–21. This text is in Newari language and in this portion the paper orientation is different from the rest folios of the MS.
