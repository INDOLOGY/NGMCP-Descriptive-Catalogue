Manuscript Features

This MS contains portions of the third, fourth and sixth adhyāya of the Cāndravṛtti. The foliation starts with the first pāda of the third adhyāya (fols. 1-3 are lost). The extant folios are: 4-8; 10-11; 13-14; 18; 22; 25-26; 28; 30-33; 35-39; 41; 43-44; 46-47; 49. There is one more folio, written in the same hand and numbered “34”, which contains the commentary on sūtras 6.1.70-75. The actual order of folios on the microfilm is as follows: ⁅2⁆2; 4-8; 10-11; 13-14; 18; 30; 26; 25; 28; 31-33; 34; 35; 38; 49; 41; 47; 43-44; 46; 37; 39; ⁅36⁆. Some folios are slightly damaged at the margins. On a few folios, the writing has been rubbed off to some extent. There are a few corrections, probably carried out by the same hand.
