Manuscript Features

The MS contains the following texts: 1. Āryāvalokiteśvarastava 2. Āryāvalokiteśvarastavarāja 3. Śrīmadāryāvalokiteśvarabhaṭṭārakasyānanakodhastotra 4. Saptākṣarastavastotra 5. Śrīmadāryyāvalokiteśvarabhaṭṭārakasya Ślokāṣṭakastotra 6. Pañcabudhastastotra 7. Pañcākṣarastotra 8. Ṣaḍabhijñastotra 9. Dharmadhātuprabhāvāśvarabhaṭṭārakasya Ratnapūrvvāñjalistotra 10. Candraprabhāstavastotra 11. †Rakṣakāstavastuta†

The MS contains many scribal errors.
