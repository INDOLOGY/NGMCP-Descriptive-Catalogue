Manuscript Features

The first text is incomplete; available folios: 1–47, 49–63, 63–89, 91–176. The second text is also incomplete; available folios: 1–8.

The MS contains the following texts:

1. Gītātātparyapariśuddhi (on exps. 2t–178b) 2. Gītāgūḍhārthadīpikā (on exps. 183–191)

In the first text, the folio number 48 has not been assigned, but the text is continuous; the folio number 63 has been assigned twice to two successive folios; fol. 90 is missing. The second text breaks off after fol. 8.
