Manuscript Features

The MS is incomplete. Only fols. 10–56, containing the text of Śiśupālavadha 2.14 (partly), 2.15–7.69 and 7.70 (partly), are preserved. There are occasional marginal notes. As for the state of the MS, the original index card notes: “writing partly rubbed off, partly darkened”.
