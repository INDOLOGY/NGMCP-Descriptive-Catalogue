Excerpts

Beginning

śrīgaṇeśāya namaḥ

saṃsāratāpahṛdaye nidhāya

śrīramacaṃdracaraṇādvyam ādareṇa

vākyāni samyag avadhāṛyya munīśvarāṇāṃ

māṃsādinirṇayavacāṃsi vicārayāmi

yenācāraḥ sutām uktaḥ sarvavarṇāśramāgataḥ

ādyo dhipālo lokānām ājñāta(svā)śituṃ palam (fol. 1v1–3)

End

ity āstāṃ vistaraḥ pravarttako vāpi nivarttako vā (jalaṃkṛtavratasyekasyāropya) hema‥ yad api tathāpi vidyārasikaḥ prayatnam enaṃ samālokayituṃ yata (śālāva‥ntām) yukta (fol. 7v1–3)

Colophon

iti madanamanoharaviracitāyāṃ māṃsapīyūṣalatāyāṃ saptama[ḥ] stavakaḥ samāptaḥ nījanābhyaṃtabhūṣaṇamaṇivrajavittaḥ || (fol. 7v3–4)
