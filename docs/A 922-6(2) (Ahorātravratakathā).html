Manuscript Features

The manuscript contains a metrical version of the Ahorātravratakathā. In the text appear the names of Subhūti, Vajradhara, Śāriputra, Dharmadakṣa and Nirghoṣadamana. The content seems to be the same as the prose version of the Ahorātravratakathā edited by Handurukande (Handurukande 2000, pp. 83-101).

Available folios are nos. 1-5. They are dispersed in the manuscript. Fol. 3-5 have got a wrong number (4, 5, and 1 respectively).
[A 922-6] exposure 9b → 10a → 10b → 11a → 19b → 20a → 15b → 16a → 11b → 12a
[A 119-3] exposure 31a → 30b → 30a → 29b → 7b → 8a → 25a → 24b → 29a → 28b

The text is corrupt in many places.
