Manuscript Features

The writing is of variant size and appearance and was possibly done by various hands. The reading is corrupt in many parts of the MS (for a carefully written MS of the same text confer A 52/11).

Fol. 3 has been filmed between fols. 5 and 6.

There is a short description of this MS in BSP vol. vi p. 28 no. 98.
