Manuscript Features

The manuscript consists of parts of two originally independant manuscripts. Both are of about the same age. The first part was filmed in the wrong order with the last folio first. The contained (excerpts of) texts in the order on the microfilm are:

  • 1st part:
    • C 44-8(4) Dakṣasmṛti
    • C 44-08(3) Bhāradvājasmṛti(1)
    • C 44-8(2) Vyāsasmṛti(1)
    • C 44-8(1) Bṛhaspatismṛti(1)
  • 2nd part:
    • C 44-8(5) Bṛhaspatismṛti(2)
    • C 44-8(6) Smṛtisamuccaya
    • C 44-8(7) Dharmasārasamuccaya
    • C 44-8(8) Bhāradvājasmṛti(2)
    • C 44-8(9) Vyāsasmṛti(2)
    • C 44-8(10) Parāśarasmṛti
    • C 44-8(11) Saṃvartasmṛti
    • C 44-8(12) Śātātapasmṛti
