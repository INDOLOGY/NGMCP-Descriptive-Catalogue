Manuscript Features

This manuscript is written by several hand. As the content consists mainly in mantras, it is not easy to determine if folios are missing or if the extant folios did even belong to various manuscripts. The verso of the last folio (no. 17?) is illegible except for the last two lines. Only the first folio has been numbered.

At the end one folio of another manuscript is found. Only one side of it has been filmed. It begins: oṃ namaś caṇḍikāyai || malasodhanasnānaṃ karttavyam ||
